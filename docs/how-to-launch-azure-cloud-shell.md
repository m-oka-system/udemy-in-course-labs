## Azure Cloud Shell の起動

1. Azure ポータル右上の [Cloud Shell] アイコンをクリックします。
2. Azure ポータル下半分に Cloud Shell 画面が起動したら [Bash] をクリックします。
3. 以下のとおり選択して [適用] をクリックします。

   - ストレージアカウントは不要です (既定値)
   - プルダウンからサブスクリプションを選択

   <img width="798" height="277" alt="Image" src="https://github.com/user-attachments/assets/07dfb127-c299-4ded-80ad-65521aad5bca" />

4. ターミナル画面が起動することを確認します。
