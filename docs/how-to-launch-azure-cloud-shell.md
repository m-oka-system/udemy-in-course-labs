## Azure Cloud Shell の起動

1. Azure ポータル右上の [Cloud Shell] アイコンをクリックします。
2. Azure ポータル下半分に Cloud Shell 画面が起動したら [Bash] をクリックします。
3. 以下のとおり選択して [適用] をクリックします。

   - ストレージアカウントは不要です (既定値)
   - プルダウンからサブスクリプションを選択

   ![Azure Cloud Shell1](https://i.gyazo.com/9d451ec7f7cb3ee4db3675b4a06c6dde.png)

4. ターミナル画面が起動することを確認します。

   ![Azure Cloud Shell2](https://i.gyazo.com/f8af65a82bcb762f6a1aa195ce57d7f7.jpg)
