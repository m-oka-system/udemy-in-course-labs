## 🎯 Mission

Azure ポータルの外観、お気に入りをカスタマイズする方法を確認してみましょう。

## 🚀 Challenge

### 外観の設定変更

1. Azure ポータル右上の歯車アイコン [設定] をクリックします。
2. [外観] をクリックします。
3. 以下の設定を変更して [適用] をクリックします。

   - メニューの動作: ドッキング
   - サービス メニューの動作: 展開済み

4. 左側メニュー [ホーム] をクリックしてトップ画面に戻ります。

### お気に入りの追加

1. 左側メニュー [すべてのサービス] をクリックします。
2. すべてのサービスの検索欄に [サブスクリプション] と入力します。
3. 絞り込み表示された [サブスクリプション] の上にマウスカーソルを合わせて、星印のアイコンをクリックします。
4. 左側メニュー [お気に入り] の一覧の最下部に [サブスクリプション] が追加されたことを確認します。

### お気に入りの並び替え

1. 左側メニュー [サブスクリプション] をクリックしたまま、お気に入りエリアの任意の場所 (例: リソースグループの上) にドラッグ&ドロップします。
2. [サブスクリプション] メニューの並び替えができたことを確認します。

   ※希望する順序になるまでドラッグ&ドロップを繰り返します。

### お気に入りの削除

1. 左側メニュー [サブスクリプション] の上にマウスカーソルを合わせます。
2. ポップアップメニュー右上に表示される星印のアイコンをクリックして、塗りつぶされていない状態に変更します。
3. 左側メニューから [サブスクリプション] が削除されたことを確認します。
