## 🚀 Challenge

### Windows 仮想マシンの作成

1. 左側メニューの [仮想マシン] をクリックします。
2. ページの上部にあるメニューから [作成] > [仮想マシン] を選択します。

仮想マシンの作成画面は、基本情報、ディスク、ネットワークなど、設定内容ごとに複数のタブに分かれています。順番に入力していきましょう。

> [!NOTE]
> Azure ポータルのアップデート状況により、一部の項目名が英語で表示される場合があります。

#### 基本情報

3. 以下のとおり入力して [次へ] をクリックします。

| 項目                      | 設定値                                                                                   |
| ------------------------- | ---------------------------------------------------------------------------------------- |
| サブスクリプション        | 自動選択される                                                                           |
| リソースグループ          | 既存のリソースグループを選択                                                             |
| 仮想マシン名              | w-iaas-vm                                                                                |
| リージョン                | Japan West                                                                               |
| 可用性オプション          | インフラストラクチャ冗長は必要ありません                                                 |
| セキュリティの種類        | 標準                                                                                     |
| イメージ                  | [smalldisk] Windows Server 2022 Datacenter: Azure Edition - x64 Gen2                     |
| VM アーキテクチャ         | x64                                                                                      |
| サイズ                    | B2ms                                                                                     |
| Azure Spot 割引で実行する | チェックなし (既定値)                                                                    |
| ユーザー名                | 任意のユーザー名 (例: cloudadmin)                                                        |
| パスワード                | 12 文字以上で、小文字・大文字・数字・特殊文字のうち少なくとも 3 種類以上を含むパスワード |
| パブリック受信ポート      | なし                                                                                     |

#### ディスク

4. 以下のとおり入力して [次へ] をクリックします。

| 項目              | 設定値                    |
| ----------------- | ------------------------- |
| OS ディスクサイズ | イメージの既定値 (30 GiB) |
| OS ディスクの種類 | Premium SSD               |
| VM と共に削除     | チェックあり (既定値)     |
| その他の項目      | 既定値のまま              |

#### ネットワーク

5. 以下のとおり入力して [次へ] をクリックします。

| 項目                                                 | 設定値                      |
| ---------------------------------------------------- | --------------------------- |
| 仮想ネットワーク                                     | 自動生成される名前 (既定値) |
| サブネット                                           | 自動生成される名前 (既定値) |
| パブリック IP                                        | 自動生成される名前 (既定値) |
| NIC ネットワークセキュリティグループ                 | 基本                        |
| パブリック受信ポート                                 | 選択したポートを許可する    |
| 受信ポートの選択                                     | RDP (3389)                  |
| VM が削除されたときにパブリック IP と NIC を削除する | チェックあり                |
| 負荷分散のオプション                                 | なし                        |

6. 必要に応じて、[管理] [監視] [詳細] [タグ] のタブでどんな項目が設定できるのか確認して [次へ] をクリックします。
7. [確認と作成] タブで入力内容に誤りがないか確認し [作成] をクリックします。
8. デプロイが完了したら [リソースに移動する] をクリックします。
9. 仮想マシンの概要ページが開くので、リソースグループ名のリンクをクリックします。
10. リソースグループの中に以下のリソースが作成されていることを確認してみましょう。

- 仮想ネットワーク
- 仮想マシン
- パブリック IP アドレス
- ネットワークセキュリティグループ
- ネットワークインターフェース
- ディスク
